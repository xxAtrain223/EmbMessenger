Detailed Description

Base class for any commands to send to the device.

Inherit from this class and register it using EmbMessenger::registerCommand. In your constructor, be sure to set m_type_index. This is needed if you ever send it using the nontemplate version of EmbMessenger::send. Override send to send values to the device. Override receive to receive values from the device. Override reportError if your command throws custom errors.
